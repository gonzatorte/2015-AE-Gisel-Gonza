Problema de ejemplo para el curso de Algoritmos Evolutivos
----------------------------------------------------------

Teorema de los cuatro colores:

En teoría de grafos, el teorema de cuatro colores (o teorema de la minimalidad cromática) es un teorema sobre la coloración de grafos que establece lo siguiente:
"Dado cualquier mapa geográfico con regiones continuas, éste puede ser coloreado con cuatro colores diferentes, de forma que no queden regiones adyacentes con el mismo color." [1]

A modo de ejemplo, se resuelve el problema de colorear el Mapa de Montevideo usando únicamente 4 colores. Se implementan algoritmos evolutivos para resolver el problema en las bibliotecas Malva, ECJ y WatchMaker.

El mapa de Montevideo se obtuvo del Catálogo de Datos Abiertos [2]

Dentro de la carpeta correspondiente a cada biblioteca, se encuentra un README con las instrucciones para ejecutar el algoritmo, y una breve descripción de la implementación realizada. La carpeta Archivo de entrada, contiene el archivo con los límites de cada barrio, que se utiliza como entrada de los algoritmos implementados. La carpeta QGIS contiene el mapa de Montevideo, una salida de muestra, e instrucciones para poder visualizar la asignación de colores en el mapa de forma gráfica utilizando QGIS.
